# no-nofiy

🚫🙅‍♂️🙅‍♀️不要在社交网站上通知我

## 如何使用

直接在 Chrome 插件市场安装：

## 插件介绍

厌倦了各种社交网站推送给我的无聊系统通知，我在使用这些社交网站的时候，根本对于一些无效的互动，各种宣传消息没有兴趣。

所以我才写了 `no-notify` 这个插件，直接屏蔽这些社交网站推送给我的通知，我在需要读取通知的时候，会点进去看通知消息。

不算是反社交，只是想主动摄取信息，而不是被各种小红点打扰。
